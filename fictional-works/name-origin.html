オリジナルキャラクターの名前の由来について

1. 天奈あまなレティア

姓の由来

当初は「サキュバスだからみんなから愛されているだろう」ということで「愛されている」という意味のイタリア語 amata やスペイン語 amada からとって天田や甘田とするつもりでした。しかしこれらは実在するためファンタジー感が薄く感じられました。そこで後ろを少し変えて Amana や Amane のように変えようと考えました。最終的に Amana としたのは、響きがよく、また女性の悪魔ナアマ נעמה (naama)のアナグラムになっているからです。ナアマはゾーハルでは悪霊の母として描かれています。

原典 עִמֵּיהּ נָפְקַת נוּקְבָּא וַחֳדָא
転写 ʿim-mēh nāfəq-at nūqəbā va-ḥŏdā
説明 with-3ms go_out.pf-fs woman and-one.f
翻訳 そして1人の女性が彼（トバル・カイン）と一緒に産まれた。
  • וְ və- conj pfx と
  • חֲדָא ḥădā num c 1
  • נְקֵבָּה nəqebā f 女性
  • נ־פ־ק 外に出る
    • G נְפַק nəfaq 外に出る
  • עִמ ʿim prep ～と。 接尾辞が付く場合、מが二重子音化する。
  • חֵהּ -eh pron sfx 3ms
原典 נַעֲמָה וְאִתְקְרֵי בַּתְרָאָה, וְטָעָאן בִּרְיָין וַהֲווֺ
転写 naʿămā və-ʾiṯəqərē baṯərā-ʾā və-ṭāʿān barəyāy-n va-hăv-o
説明 Na’ama and-call.psv.pf.3ms after-def and-go_astray-mp creation-pl and-be.pf.3mp
翻訳 人間たちは彼女を追い求めて堕落した。彼女はナアマと呼ばれていた。
  • בְרִיָּיה bərīyāy 創造物、（神の創造物である）人間
  • הָוָה hāvā identity/existential copula（新たな話題を提示する役割ももつ）
  • טְעָא ṭāʿā 道に迷う、堕落する
  • בַּתְרַא baṯəra- prep （時間的・空間的）後で
  • ק־ר־א 呼ぶ、読む
    • G קְרָא qərā 呼ぶ
    • tG אִתְקְרֵי ʾiṯəqərē 呼ばれる
原典 אָחֳרָנִין וְשֵׁדִין רוּחִין נַפְקוּ מִינָּהּ
転写 ʾāḥŏrān-īn və-šēd-īn rūḥ-īn nafəq-ū mīn-nāh
説明 other.m-p and-demon-pl spirit-pl go_out.pf-mp from-3fs
翻訳 彼女から霊や悪霊たちが産まれた。
  • מִין mīn (מִן min) prep ～から。接尾辞がつく場合、נ が二重子音化する。
  • חָהּ -āh (חַהּ -ah) pron sfx 3fs
  • רוּחַ rūḥa 精霊、風
  • שֵׁד šed 悪霊、悪魔
  • אָחֳרָן ʾāḥŏrān (אַחֲרוֺן ʾaḥărōn) 他、次
原典 יַתִּיר. וְלָא נַטְלַת תִּיאוּבְתָּא
転写 yaṯīr və-lā naṭəlaṯ tīʾūbətā
説明 more and-neg take.pf-fs desire
翻訳 彼女は色欲を取るが、それ以上は取らない。
  • נ־ט־ל 手に取る
    • G נְטַל nəṭal 手に取る
  • תִּאוּבְתָּא tiʾūbətā 欲望
  • נ־פ־ק 外に出る
    • C אַפֵּיק ʾapēq 産み出す
  • לֹא lā neg
  • יַתִּיר yaṯīr adv より多く
原典 וְאַפִּיקַת אִתְעַבְּרַת, תִּיאוּבְתָּא וּמֵהַהוּא
転写 və-ʾapīq-aṯ ʾiṯəʿabər-aṯ tīʾūbətā ū-me-hahū
説明 and-bring_forth-3fs become_pregnant.pf-fs desire and-from-dem.ms
翻訳 その色欲によって彼女は妊娠し、
  • הַהוּא hahū それ
  • ע־ב־ר 通過する
    • tG אִתְעַבַּר ʾiṯəʿabar 妊娠する
  • נ־פ־ק
    • C אַפֵּיק ʾapēq 産み出す
